# Microsoft 365 Sign-In Outlook Toolkit

A diagnostic PowerShell toolkit for Microsoft 365 app sign-in, Outlook, and connectivity checks.

## Features

- Office Click-to-Run detection
- Installed Office app path checks
- Outlook profile registry summary
- Microsoft 365 endpoint DNS/TCP checks
- Autodiscover DNS context
- OneDrive process/path context
- WinHTTP and user proxy summary
- HTML, CSV, and JSON reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\M365_SignIn_Outlook_Toolkit.ps1
```

## Safety

Diagnostic-only. It does not clear tokens, modify profiles, remove accounts, or change registry settings.
